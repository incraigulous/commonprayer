# Common Prayer

Common Prayer is a mobile-first Progressive Web App for the 1979 Episcopal Book of Common Prayer Daily Office. It opens directly to the right prayer for the time of day with no configuration required.

## What it does

The app guides you through the four Daily Offices of the BCP — Morning Prayer, Noonday Prayer, Evening Prayer, and Compline — presenting the correct liturgy for today's date, season, and proper automatically.

Key qualities:

- **Zero friction** — tap the icon, pray. The right office for the time of day is already open.
- **Fully offline** — all liturgical content is bundled or cached locally. No network required after first load.
- **Installable** — add to your iOS or Android home screen for a native-app experience.
- **Liturgically accurate** — season detection, proper numbering, feast days, and lectionary assignments follow the BCP 1979 calendar.

## Rite options

The app supports four versions of the Daily Office:

| Version | Description |
|---|---|
| **Rite II** | Contemporary language — recommended for most users |
| **Rite I** | Traditional language ("thee", "thy", "thine") |
| **Enriching Our Worship** | Expansive language with inclusive imagery for God |
| **Daily Devotions** | Shorter form for each time of day |

Switch versions at any time in [Settings](settings.md).

## Navigation

The top tab bar gives instant access to the four primary offices:

```
[ Morning ] [ Noon ] [ Evening ] [ More ]
```

The **More** tab opens a sheet with Compline and other liturgies. The hamburger menu slides open a drawer with Prayer List, Psalter, Daily Readings, Prayers & Thanksgivings, Favorites, Reminders, Settings, and About.
