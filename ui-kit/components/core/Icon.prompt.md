A line-art icon from Lucide — thin, rounded strokes matching the manuscript
line-work. (Lucide substitutes for the production icon set; see readme.)

```jsx
<Icon name="sunrise" />
<Icon name="book-open" size="1.5rem" />
<Icon name="moon" strokeWidth={1.5} />
```

- **name** is any Lucide icon (kebab-case). Sizes in `em`/rem and inherits `color` via `currentColor`. **strokeWidth** defaults to 1.75 for the fine manuscript line.
