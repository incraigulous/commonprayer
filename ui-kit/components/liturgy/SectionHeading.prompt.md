A display-serif heading with the short rubric-red rule that opens each part
of the office.

```jsx
<SectionHeading level="display" eyebrow="Proper 9">Morning Prayer</SectionHeading>
<SectionHeading>Confession of Sin</SectionHeading>
<SectionHeading level="office" rule={false} center>The Lessons</SectionHeading>
```

- **level**: `display` (office title) · `office` · `section` (parts). **eyebrow** adds a small-caps kicker; **rule** toggles the rubric underline; **center** centers.
